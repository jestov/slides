Gestor de recursos
de interfaces gráficas

¿Qué es?

UI BOX permitirá guardar recursos de una interfaz de usuario o incluso de toda una marca, todo esto dividido por proyectos.

Desarrollo de software por etapas.

  1. Plan operativo.
  2. Especificación de requisitos.
  3. Especificación funcional.
  4. Diseño. <-
  5. Implementación.
  6. Integración.
  7. Validación y verificación.
  8. Mantenimiento.

Investigación.

  1. Curso de VueJS: Componentes de Vue.
  2. Curso de NodeJS: Express JS, importación y exportación de módulos.
  3. Curso MongoDB: Conexión a la DB, creación y manipulación de las bases de datos, estructura de la DB (JSON).
  4. Curso MEVN Stack: Creación de aplicación CRUD.

Alcances.

  2. Login.
  3. Creación de folders (proyectos).
      4. Dentro de los folders:
